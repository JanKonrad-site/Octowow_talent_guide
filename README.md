# OctoWoW Talent Calculator CZ

Statická česká talent kalkulačka pro GitHub Pages.

## Obsah

- `index.html`
- `style.css`
- `script.js`
- `data.js`
- `.nojekyll`

## Použití na GitHub Pages

Nahraj celý obsah složky do repozitáře a v GitHubu zapni Pages pro větev `main` a root složku.

Profily se ukládají pouze lokálně v prohlížeči přes `localStorage`. Export/import funguje přes JSON. Neobsahuje žádný backend ani serverové ukládání.
